# Проект по автоматизации тестирования "Яндекс.Самокат"

Привет! Это учебный проект для 6-го спринта в Яндекс.Практикуме.
Я написал UI-тесты для сайта "Яндекс.Самокат".

## Что я использовал:
- `Python`
- `Selenium`
- `Pytest`
- `Allure Pytest` для отчетов

## Какие тесты я написал:

1.  **Проверка раздела "Вопросы о важном"**
    - Тест проверяет, что при клике на вопрос появляется правильный ответ. Сделано через параметризацию, чтобы не дублировать код.

2.  **Проверка заказа самоката**
    - Тест проходит весь путь заказа от начала до конца.
    - Используются два разных набора данных (имя, адрес и т.д.) через параметризацию.
    - Проверяются обе кнопки "Заказать" — и вверху, и внизу страницы.

3.  **Проверка кликов по логотипам**
    - Тест для логотипа "Самокат": проверяет, что он возвращает на главную страницу.
    - Тесты для логотипа "Яндекс": проверяют, что клик открывает новую вкладку, и что эта вкладка ведет на нужный сайт.

## Особенности проекта:

*   **Page Object:** Все локаторы и методы работы со страницами вынесены в отдельные классы (`/pages` и `/locators`), чтобы тесты были чистыми и читаемыми.
*   **Стабильность:** В тестах используются явные ожидания (`WebDriverWait`), чтобы избежать случайных падений из-за скорости загрузки.
*   **Тест на логотип "Яндекса":** Я заметил, что переход на Дзен может быть нестабильным. Поэтому я разделил эту проверку на два небольших, но очень надежных теста. Один проверяет открытие новой вкладки, а второй — что ссылка ведет куда нужно. Так тесты не будут "моргать" (flaky).

## Как запустить проект:

1.  **Установите зависимости:**
    ```bash
    pip install -r requirements.txt
    ```
2.  **Запустите тесты:**
    ```bash
    pytest -v --alluredir=allure-results
    ```
3.  **Посмотрите отчет:**
    ```bash
    allure serve allure-results
    ```
